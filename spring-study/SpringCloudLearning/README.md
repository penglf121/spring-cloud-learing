# Spring Cloud学习与实践

------
>Spring Cloud是一个基于Spring Boot实现的微服务架构开发工具。它为微服务架构中涉及的配置管理、服务治理、断路器、智能路由、微代理、控制总线、全局锁、决策竞选、分布式会话和集群状态管理等操作提供了一种简单的开发方式。

Spring Cloud包含了多个子项目，本人学习并实践了一部分模块，具体内容可看下面文章，文章与代码也会持续更新。
